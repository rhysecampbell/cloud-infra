Rest quick guide:

/rest/v1.0
   /get
      /list
        POST data: regionID, stationName, xmlName  
      /listStationMap
      /xmlName
      /stationName
      /regionID
   /getSimple/
      Examples: 
      /rest/v1.0/getSimple/stationNameSimple
      /rest/v1.0/getSimple/xmlNameSimple
      /rest/v1.0/getSimple/userListSimple
   /user
      /list/
      /listTable
      /listSimple/
      /listRoles/  (Used to return the user's roles)
      /add  (Add a new user)
        POST data: username, roles[], password, addedBy, description
      /setRole (Set the user's roles)
        POST data: username, roles[]
      /setRoleProperties
        POST data: roleID, metarData, ltgData
      /setPassword  (Change the users's password)
        POST data: username, password
      /delete  (Delete the user with his given user ID)
        POST data: userID
